# Paper Store

A dummy paper store application built with Node.js, Express, and PostgreSQL. It allows users to save research papers by their DOI, view saved papers, and delete them. The application also includes a search interface for querying external APIs, too.

This project is intended for demonstration and testing purposes, showcasing basic CRUD operations, database interactions, and a simple frontend interface. It is not meant for production use and lacks authentication, validation, and security features.
